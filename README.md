# 🎯 Tic Tac Toe Game

This project is a Tic Tac Toe game built using HTML, CSS, and JavaScript. It provides a clean and interactive interface where two players can take turns placing X’s and O’s on a 3×3 grid. The game includes logic to detect wins, draws, and automatically resets for replay. This project was created to practice JavaScript game logic, DOM manipulation, and responsive UI design.
